EEG Graphing Demo

Welcome to the EEG Graphing Demo. Follow the steps below to visualize EEG data:

  1. Click the "Start Data Stream" button to begin simulating EEG data.
  2. Click the "Stop Data Stream" button to stop the data simulation.
  3. To use your own data, prepare your EEG data in a CSV file format.
  4. The first row should contain time stamps, with subsequent rows containing EEG channel data.
  5. Upload the CSV file using the input below.
  6. Once uploaded, the time series, FFT plot, and head plot will update to reflect your data.

Example CSV format:

Time,Channel1,Channel2,Channel3,Channel4,Channel5,Channel6,Channel7,Channel8
0,0.5,1.2,0.8,0.9,1.1,0.7,0.6,0.4
1,0.7,1.4,0.6,0.8,1.3,0.8,0.5,0.3
2,0.9,1.6,0.7,0.7,1.5,0.9,0.4,0.2
