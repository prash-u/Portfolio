Gene Expression Analysis Tool

Welcome to the Gene Expression Analysis Tool. Follow the steps below to visualize your genetic data:

  1. Prepare your gene expression data in a CSV file format.
  2. The first row should contain gene names, with the first cell left empty.
  3. The first column should contain sample names, with the first cell left empty.
  4. Upload the CSV file using the input below.
  5. Once uploaded, a heatmap will be generated to visualize the gene expression profiles.

Example CSV format:

Sample,Gene1,Gene2,Gene3
Sample1,0.5,1.2,0.8
Sample2,0.7,1.4,0.6
Sample3,0.9,1.6,0.7
